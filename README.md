# Secure SSH Access Automation

This project demonstrates secure SSH key-based authentication, Linux server hardening, Ansible-based SSH key distribution, GitHub/GitLab SSH setup, and CI/CD deployment using SSH keys.

## Tools Used
- Linux
- OpenSSH
- GitHub / GitLab
- Ansible
- GitHub Actions

## Project Goals
- Automate SSH key generation
- Configure passwordless SSH login
- Distribute public keys using Ansible
- Use SSH keys for Git operations
- Use SSH keys securely in CI/CD pipelines
